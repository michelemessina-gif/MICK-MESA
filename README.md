# Mick Mesa

A full-stack artist platform built as part of my software engineering journey.

## Overview

Mick Mesa is a personal artist website designed to showcase an original music project, album concepts, future tour plans, merchandise concepts, and a direct contact system.

The project began as a frontend website and is being progressively expanded into a full-stack application.

---

## Features

### Frontend

- Responsive design
- Cinematic hero section
- Album showcase
- Tracklist section
- Artist story section
- Support section
- Tour section
- Store preview
- Contact form

### Backend

- Node.js server
- Express API
- Contact form endpoint
- JSON request handling
- Form validation
- API responses

## Current Architecture

User submits contact form

Frontend JavaScript
↓
fetch()
↓
POST /api/contact
↓
Express Route
↓
Validation
↓
Response
↓
Frontend updates UI
